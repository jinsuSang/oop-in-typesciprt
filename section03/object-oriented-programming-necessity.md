# 개체지향 프로그래밍의 필요성

## 구조체의 한계

- C 언어에서 데이터의 그룹화로 새로운 데이터 형을 만드는 방법이다 

- 최근에는 golang 언어에서 사용한다 

  ```go
  type Human struct {
      name string
      age  int 
  }
  
  func (h Human) getName() string {
      return h.name
  }
  ```

- 구조체의 한계
  - 데이터와 동작이 분리되어 있다 
  - 어떤 구조체가 어떤 함수랑 연관 있는지 찾기 힘들다

## 세상을 인지하는 법

- 개체는 상태와 동작을 가지고 있다 
-  모든 것이 개체는 아니다
